# Lab 3 — Cloud Architecture & Migration Planning

## Objective
Design cloud architecture and migration approach.

## Tasks
1. Design a 3-tier cloud architecture:
   - Web layer
   - App layer
   - Database layer

2. Select cloud components:
   - Compute
   - Storage
   - Network
   - Security

3. Create a migration plan for:
   “College ERP system moving to cloud”

4. Define:
   - Migration phases
   - Risks
   - Rollback plan

## Tools
Draw.io / Lucidchart

## Submission
Upload architecture diagram + migration document to GitHub.
